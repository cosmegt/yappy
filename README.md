# Yappy!
A web messaging app that welcomes new (faster) frameworks.
## Why
Because this technologies have entered the web and creating new tools that implement them is important for the creators, for the projects, and for the users.

## What new technologies
+ [Svelte](https://svelte.dev/) alternative to popular React, Angular, or Vue.
+ [GraphQL](https://graphql.org) alternative to popular REST APIs.
+ and more to come!
